About Prasis

Prasis comes from the Greek word πρᾶσις (prásis) — “sale”. This site is a semester project front-end for the Noroff Auction API (v2).

Anyone can browse and search listings. Registered users (with @stud.noroff.no) can create listings, upload images, set deadlines, and place bids using credits.

  • Create listings with title, deadline, images, and description
  • Bid on other users’ listings and view bid history
  • Browse and search without logging in
